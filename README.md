## 🚀 How to Run

### Step 1 — Install Python (if not installed)
Download from https://www.python.org/downloads/

### Step 2 — Install Flask
Open terminal / command prompt in the project folder:
```bash
pip install flask
```

### Step 3 — Run the app
```bash
python app.py
```

### Step 4 — Open in browser
Visit: **http://127.0.0.1:5000**

## 🧠 How It Works

1. **User Input** — User pastes the suspicious text
2. **Text Analysis** — System cleans and analyzes the text for keywords, patterns, and signals
3. **AI Model** — Logistic regression model evaluates scam probability
4. **Prediction Result** — Fraud Risk Score (0–100%) + Verdict (SCAM / SUSPICIOUS / LEGITIMATE)
